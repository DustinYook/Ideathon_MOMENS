# **MOMENS**

#### 기획의도 : 문장을 입력하면 몇 단어로 구성되어있는지 단어의 빈도는 어떤지 표시해주는 역할을 하게 함
------

## 추가적으로 개선할 사항

#### 1. 폰트 적용

#### 2. 부트스트랩 예제 및 컴포넌트 적용

#### 3. 기존의 단어 수만 세는 것 외에 공백을 제외하거나 포함한 총 글자 수를 세어줌
